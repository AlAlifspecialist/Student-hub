# 4-Minute Contribution Video Outline

## 1. Introduction (~30 sec)
- Introduce the feature you built.
- Explain why it matters in the project.

## 2. Technical Explanation (~90 sec)
- Show the related GitHub commits.
- Explain database changes, PHP pages, and any security decisions.
- Mention one challenge and how you solved it.

## 3. Demonstration (~90 sec)
- Show the feature working in the browser.
- Briefly point to the matching code files.

## 4. Impact Summary (~30 sec)
- Explain how the feature improves the project.
- Mention 1–2 future improvements.
